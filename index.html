Q-1:How does javascript work?

সহজ ভাষায় বলতে গেলে জাভাস্ক্রিপ্ট হলো একটি interpreted language. C++ অথবা java এর মত এটি compiled language নয়। compiled language সরাসরি machine code এ converted করা থাকে যেটি প্রসেসর execute করতে পারে। interpreted language এ কোড line by line execute করা হয়। javascript একটি interpreted language অর্থাৎ javascript এ কোড line by line execute হয়। যখন browser page load করে তখন browser এর built-in interpreter কোডটি run করায়। javascript হলো একটি single threaded এবং single concurrent programming language. javascript হলো asynchronous programming language. asynchronous programming দ্রুত execution এর জন্য ভালো। এছাড়াও javascript কে lightweight programming language ও বলা হয়।

Q-2:What is the differences between Local Storage & Session Storage?

  • Local storage হলো javascript site এ কোন ডেটা কে key-value pair হিসেবে সংরক্ষন করা, যেটি browser বন্ধ করে দিলেও browser এ থেকে যায়।
  • Session storage এ ডেটা name-value ভাবে save করা থাকে। Session storage এর ডেটা কিন্তু Local storage এর মত browser বন্ধ করে দিলেও saved থাকেনা, ডেটা মুছে যায়।
  • Local storage এর accessibility হলো যে কোন window.
  • Session storage এর accessibility হলো same tab.
  • Local storage এর ডেটা expiry থেকে যায়।
  • Session storage এর ডেটা tab close করলেই মুছে যায়।
